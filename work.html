Full tier breakdown

  • Core — $2,500

    Logo (primary lockup), color palette with codes, typography stack, 1-page brand sheet, SVG + PNG file delivery.

  • System — $4,500 Most projects land here

    Full logo suite (primary + alternates), extended color system, typography hierarchy, usage guidelines, complete file suite.

  • Book — $8,000

    Everything in System + naming, brand voice/tone, application mockups (4–8 real-world scenes), full Brand Book PDF.
