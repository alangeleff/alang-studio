Brand Identity Origination

Full tier breakdown

  • Proof — $500

    A concept pass before you commit to the full identity.

    • 1 wordmark concept
    • 1 color direction
    • 1 look-and-feel comp
    • Offsets any upgrade — pay the remaining balance if you proceed

  • Core — $2,500

    A finished, usable identity for founders who need to ship.

    • Naming — business name research + recommendation
    • Wordmark / logo (primary lockup)
    • Color palette (HEX/RGB/CMYK codes)
    • Typography stack (headline + body fonts, hierarchy + download links)
    • Favicon + social avatar set (all major platforms pre-sized)
    • Brand color + type reference card (1-page PDF — swatches, font names, download links)
    • Brand Sheet (1 polished page: logo, palette, fonts, tagline)
    • File delivery — SVG + PNG suite

  • System — $4,500 Most projects land here

    A complete brand system — usable, scalable, ready for a team.

    Everything in Core, plus:

    • Full logo suite (primary, alternate, horizontal, stacked, icon-only lockups)
    • Extended color system (primary + secondary + neutrals)
    • Typography hierarchy with real examples
    • Competitor positioning snapshot (1-page: 2–3 competitor visual audits + how you stand apart)
    • Email signature template (branded HTML)
    • Business card mockup (1 rendered scene)
    • PWA included when bundled with a website build
    • Brand Guide (6–10 pages: full logo suite, colors, typography, positioning, usage do's/don'ts)

  • Book — $8,000

    The full identity treatment — voice, story, application, and a bound brand book.

    Everything in System, plus:

    • Brand voice + tone guidelines (written personality, tone, what the brand never says)
    • Application mockups (4–8 real-world scenes: business card, phone, social header, website, merch, etc.)
    • Animated logo (delivered as .svg, .gif, .mp4 — lives in asset kit, not the PDF)
    • Custom icon set (5–8 brand-specific icons in the brand's visual language, SVG + PNG)
    • PWA included when bundled with a website build
    • Brand Book (15–20 pages: brand story, voice/tone, mockup scenes, full usage system)

Every tier includes name & domain research — availability check by state plus domain options, presented for you to decide.
